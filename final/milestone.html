Final Project Proposals: Nerf Pathtracer with diverse lighting



Overview

We will introduce NRTL, Neural RayTracing with diverse Lighting, which extend the relighting capability of Neural RayTracing from point light to more a more diverse set of lights including area lights and directional lights.

Progress Report

We implement a RTX based Nerf pipeline according to literature. We test our preliminary implementation on the standard lego scene from NERF dataset. We pretrain our model using a standard VolSDF for 50000 epoches and futher train our ray-traced model for 25000 epochs. We used 2 Nvidia V100 to complete this task. We visualize results as follows

Preliminary discussions

As shown in the figure, we achieve visually satisfactory results. However there are still serveral rooms for improvement.

1. Optimization time. Our current model takes 12 hour to train, we will explore ways to accelerate Optimization

2. Visual Details: the model lack high frequency details. We will consider bororrow some ideas from state-of-the art such as fourier embedding to imporve the details of rendered scene.

3. Additional light source. Since we explict represnt the scene using a rendering equation, we can add arbritary lights to the scene. We will explore environment map as well as area lights.
